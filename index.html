Relative Values:

We can divide it into three consicutive parts which are ('Parent's dimension(%)','the current font size(rem,em,ex)','the viewport dimensions(vh,vw)')

% value

percentage value relative to any element for example here the div element take 15% of the body's width

vh&vw values

(VH)and(VW) are similar to the percentage value relative to the viewport for example here the div element take 30% of the screen's width and height

em value

EM is relative to the font size of the parent element (Body)

rem value

REM is relative to the font size of the root element (div)
